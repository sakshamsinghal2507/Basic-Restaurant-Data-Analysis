# 🍽️ Basic Restaurant Data Analysis

## 📌 Project Overview
This project analyzes restaurant customer data to understand spending patterns, tipping behavior, and customer distribution using Python.

## 🛠️ Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Analysis Performed
- Customer distribution (Male vs Female)
- Daily revenue trends
- Relationship between total bill and tip
- Correlation between numerical features (Heatmap)

## 📈 Key Insights
- Higher total bills tend to result in higher tips
- Certain days generate more revenue
- Customer distribution varies by gender
